DZ1 obuhvata obradu signaala furijeovom analizom.

Kako pokrenuti:
Potrebno je imati latex podrsku za jupyter

     apt-get update && apt-get install -y texlive-latex-base texlive-fonts-recommended texlive-fonts-extra dvipng cm-super

Ostale biblioteke za python su uredno unešene u koodu.
oe3dosRequirements je sa biblioteekama potrebnim za vežbe iz predmeta DOS koje sam takodje ubacio jer sam ih instalirao za svaki slučaj.

Kako bi pokrenuo kood naa Macbook-u napravio sam dockerfile koji se takodje nalayi u ovom direktorijumu.
